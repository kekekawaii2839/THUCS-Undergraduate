# THUCS-Undergraduate
虽然这个repo叫THUCS，但实际上对应的培养方案为计算机与金融双学士学位项目，部分课程与贵系并不一致，参考时请注意！

这个repo包含的资料很杂，有各位学长学姐传下来的考题，也有老师的ppt，还有本人留存下来的考题…

在thu碰到了很多热心的学长学姐，故建此repo，希望能帮到以后的学弟学妹们！
