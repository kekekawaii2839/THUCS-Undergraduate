题目描述

小明在计算函数值时发现很多函数都可以使用C++来实现，而cmath头文件中提供了大量实用的数学函数，在做数学题的他想偷偷懒，于是编写了一个函数类，并且自己实现了一个指数函数类。他现在又想偷懒了，于是找到了你，希望你完善一次函数、组合函数、复合函数的实现。

样例解释：

这个样例构造了一个经过两次复合得到的函数

第一行3表示最终我们要的函数我们记作 h ( x ) h(x) h(x)，他由 f ( x ) f(x) f(x) 和 g ( x ) g(x) g(x) 复合而成， h ( x ) = f ( g ( x ) ) h(x)=f(g(x)) h(x)=f(g(x))

第二行表示我们构造内层的函数 g ( x ) g(x) g(x)，他是一个复合函数由下面4行的函数复合而成

记 g ( x ) = u ( w ( x ) ) g(x) = u(w(x)) g(x)=u(w(x))

第3、4行构造了一个线性函数 w ( x ) = 1.2 x + 3.3 w(x)=1.2x+3.3 w(x)=1.2x+3.3

第5、6行构造了一个指数函数 u ( x ) = 5. 2 x u(x)=5.2^x u(x)=5.2x

复合得到的结果即 g ( x ) = u ( w ( x ) ) = 5. 2 1.2 x + 3.3 g(x)=u(w(x))=5.2^{1.2x+3.3} g(x)=u(w(x))=5.21.2x+3.3

第7、8行构造了一个线性函数 f ( x ) = 1.6 x + 2.8 f(x)=1.6x+2.8 f(x)=1.6x+2.8

最外层复合的结果即为 f ( g ( x ) ) = 1.6 × ( 5. 2 1.2 x + 3.3 ) + 2.8 f(g(x)) = 1.6 \times ( 5.2 ^{1.2 x+3.3}) + 2.8 f(g(x))=1.6×(5.21.2x+3.3)+2.8
