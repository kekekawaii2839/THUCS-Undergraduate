小肖的饭店

小肖的饭店

问题描述

小肖是一家大饭店的经理，这家饭店共有 C C C 位大厨（编号从 0 0 0 至 C − 1 C-1 C−1），每位大厨都有一道自己擅长的招牌菜。

小肖每天要处理许多订单。方便起见，我们假设每个订单只包含一道菜品。当小肖收到一个订单后，他会优先安排擅长该菜品的大厨制作（如果没有，再安排其他大厨制作）。如果一个订单有多位可选的大厨，小肖会选择编号更小的那位。

特别地，一些订单指定了大厨编号，小肖必须安排指定的大厨来制作这些订单，无论他们是否擅长这些菜品。

小肖想知道，在给定所有已知信息的情况下，每个订单都会由哪些大厨制作。

要求

给定 main.cpp 以及 chef.h（下载链接），请你实现如下文件：

  • manager.h：根据 main.cpp 的接口调用，确定经理类 Manager 的接口；
  • manager.cpp：根据 manager.h 的接口设计，实现经理类 Manager；
  • chef.cpp：根据 chef.h 的接口设计，实现大厨类 Chef；
  • Makefile：使用 make 命令必须要能生成可执行文件 main。

你不能修改我们给定的 main.cpp 和 chef.h，也不必要提交它们。在编译运行时，我们会为你自动补充这两个文件。

请将你的文件打包成一个 zip 格式的压缩包并上传。注意：你的文件应该在压缩包的根目录下，而不是压缩包的一个子文件夹下，换而言之，解压你提交的压缩包后，应该直接得到上述四个文件，而不是一个包含它们的文件夹。

输入格式

第 1 1 1 行一个正整数 C C C，表示大厨的数量。

接下来 C C C 行，依次描述每位大厨。其中第 i i i 行描述编号为 i − 1 i-1 i−1 的大厨，包含两个由小写字母组成的字符串，分别表示该大厨的名字及其擅长的菜品名称。

接下来 1 1 1 行一个正整数 N N N，表示订单数量。

接下来 N N N 行依次描述所有订单，每行的开头是一个整数 t ∈ { 0 , 1 } t\in\left\{0, 1\right\} t∈{0,1}，表示该订单没有指定大厨（ t = 0 t=0 t=0）或指定了大厨（ t = 1 t=1 t=1）。接下来是一个小写字母组成的字符串，描述订单对应的菜品名称。如果该订单指定了大厨（ t = 1 t=1 t=1），则接下来是一个额外的非负整数 i ∈ { 0 , 1 , ⋯ , C − 1 } i\in \left\{0, 1, \cdots, C-1\right\} i∈{0,1,⋯,C−1}，表示指定的大厨编号。

输出格式

对于每个订单，依次输出处理结果：如果订单由一位擅长该菜品的大厨制作，输出 {chef} made an awesome {dish}；否则，输出 {chef} made a plain {dish}。这里 {chef} 和 {dish} 分别表示大厨的名字和菜品的名称，详见【样例输出】

样例

样例输入

样例输出

alice made a plain beautifulchicken
carol made an awesome friednoodles
carol made a plain kungpaochicken

提示

  1. 在调试样例时，请善用 diff（对于 Linux、MacOS、WSL 等类 UNIX 终端）或 fc（对于 Windows 原生命令提示符）命令进行输出的比对，避免输出格式问题。
  2. 由于你不能修改 chef.h，并且我们没有声明 Chef 类的默认构造函数，因此你可能需要使用一些避免调用 Chef 类默认构造函数的方法来存储 Chef 对象，如借助 std::vector 等。
